# QuizMaster (ALX FE Capstone)

A responsive quiz web application that allows users to configure and take quizzes, then view results and review answers.

## Features
- Quiz Setup: category, difficulty, number of questions
- Quiz Flow: one question at a time, shuffled options, progress indicator
- Timer: countdown for the whole quiz and auto-submit on timeout
- Results: score, percentage, performance label
- Review Answers: correct vs selected answers for each question
- UI States: loading, error, empty (no questions)

## Tech Stack
- React (Vite)
- React Router
- Tailwind CSS
- Open Trivia Database (OpenTDB)
