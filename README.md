
# QuizMaster (ALX FE Capstone)

A responsive quiz web application that allows users to configure and take quizzes, then view results and review answers.

## Tech Stack
- React (Vite)
- React Router
- Tailwind CSS
- Open Trivia Database (OpenTDB)

## Core Features (MVP)
- Quiz Setup: category, difficulty, number of questions
- Quiz Flow: one question at a time with multiple-choice answers
- Timer: countdown for the whole quiz (planned)
- Results: score + percentage + performance label
- Review: correct vs selected answers

## Getting Started
```bash
npm install
npm run dev
